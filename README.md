# Control

O "Control" é um aplicativo GUI desenvolvido em Python para agendar o desligamento ou reinício do computador. Com uma interface simples e intuitiva criada com Tkinter, você pode facilmente definir o tempo em segundos ou escolher uma das opções disponíveis para o agendamento. Além disso, você tem a opção de cancelar a ação se necessário.

## Funcionalidades

- **Agendar Desligamento/Reinício:** Insira o tempo desejado em segundos ou selecione uma das opções fornecidas.
- **Confirmar Ação:** Após definir o tempo, confirme a ação e o aplicativo cuidará do restante.
- **Cancelar Ação:** Se mudar de ideia ou inserir um valor incorreto, clique em "Cancelar" para interromper o agendamento.

## Requisitos

1. **Python:** Certifique-se de ter o Python instalado em seu sistema.
2. **Bibliotecas:** Execute o arquivo `install_libraries.cmd` para instalar as bibliotecas necessárias. Durante a instalação, será oferecida a opção de iniciar o aplicativo automaticamente.
3. **Execução do Aplicativo:** Após a instalação, você pode iniciar o aplicativo executando o `run_script.vbs`.

## Instruções

1. **Instalação:**
   - Execute o `install_libraries.cmd` para instalar as bibliotecas necessárias.
   - Siga as instruções na tela para iniciar o aplicativo, se desejado.

2. **Execução:**
   - Após a instalação, você pode iniciar o aplicativo manualmente clicando duas vezes no arquivo `run_script.vbs`.

## Notas

- O botão "Cancelar" é destinado exclusivamente para interromper qualquer ação de desligamento ou reinício programada.
- Certifique-se de que o tempo inserido está correto para evitar o agendamento indesejado.
